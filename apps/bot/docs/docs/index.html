Welcome to LondonTransit

LondonTransit is a Discord bot that brings London's essential transport network to your server. With support for the entire tube network as well as DLR, Overground, the Elizabeth line and support for buses coming soon, You can navigate London in the comfort of your community, or tiny friend group.

Supported Services

  • Tube

    LondonTransit supports the entire London Underground network, with service status and journey planning.

  • Light rail

    LondonTransit supports the DLR and Croydon Tramlink with service status and journey planning.

  • Heavy rail

    LondonTransit supports the London Overground and Elizabeth line with service status and journey planning.

  • Cycle hire

    LondonTransit supports the Santander Cycles cycle hire scheme with live availability information for docking stations.

  • Buses

    Support for the bus network is coming soon.

Technical Information for nerds

Languages

LondonTransit is written in JavaScript, using the node.js runtime framework.

Acknowledgements

LondonTransit uses the TfL Open Data API to provide the majority (currently all) of the service information provided.

LondonTransit uses the axiosnode.js library for API requests and the discord.js library for the Discord API.
